# udisks2-qt6

UDisks2 DBus interfaces binding for Qt6.

### Build dependencies

- cmake
- qt6-base-dev
- pkg-config

## Installation

### Build from source code

``` shell
$ cd /path/to/your/source/code
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```
